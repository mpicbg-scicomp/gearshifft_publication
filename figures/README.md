# figures

This directory contains the scripts to produce the figures used in this publication. Please build the all target before compiling the entire publication:

```
$ make
```

# dependencies

## R, 3.2.3+

with packages:
    - docopt
    - ggplot2
    - readr
    - dplyr
    - gridExtra
    - scales

## inkscape

for the conversion from svg to tex
